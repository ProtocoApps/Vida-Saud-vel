# INSTRUÇÕES PARA ATIVAR GITHUB PAGES

## Passo 1: Acesse as configurações do repositório
1. Vá para: https://github.com/ProtocoApps/Vida-Saud-vel
2. Clique na aba "Settings" (Configurações)

## Passo 2: Configure o GitHub Pages
1. No menu lateral esquerdo, clique em "Pages"
2. Na seção "Build and deployment", em "Source", selecione:
   - "Deploy from a branch"
3. Em "Branch", selecione:
   - Branch: "main" 
   - Pasta: "/ (root)"
4. Clique em "Save"

## Passo 3: Aguarde o deploy
- O GitHub vai levar 2-5 minutos para fazer o deploy
- Após o deploy, seu app estará disponível em:
  https://protocoapps.github.io/Vida-Saud-vel/
